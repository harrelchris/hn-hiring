# Hackernews Hiring

Extract useful insights from monthly Hackernews Who Is Hiring threads.

## Install

```shell
git clone https://github.com/harrelchris/hn-hiring
cd hn-hiring
python -m venv .venv
.venv\Scripts\activate
pip install --editable .
```

## Quickstart

```shell
hn download 36152014
hn extract 36152014
```
